
# Apply inline styles

Assume you have a DOM element `el`.

  1. `el.style.color = 'tomato';`
  2. `el.style.setProperty('color', 'tomato');`
  3. `el.style.cssText = 'color: tomato';`
  4. `el.setAttribute('style', 'color: tomato');`
